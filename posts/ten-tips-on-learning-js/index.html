Ten Tips on Learning JavaScript

Posted May 5, 2019

Below are outlines on Whatsdev Tenzin’s video with the same title.

  1. Learn the basics
  2. Solve algorism challenges
    • doing them along the way while you learning other things
  3. Learn basics of computer science - CS 50 by Harvard
  4. Go through one of the books
    • you don’t know JavaScript
    • effective JavaScript
    • JavaScript design patterns
    • JavaScript the good parts
  5. Things must-know
    • Async JavaScript and DOM
    • timer
    • callback
    • async, await
    • IIFE, closure
  6. Build projects
  7. OOP
  8. Functional programming
  9. Learn more design patterns, MVC
  10. Learn one framework and stick with it, Vue.js or React

Other tips:

  • Learn testing
  • Learn other different languages if you can
  • Keep learning
  • Teach what you learn
